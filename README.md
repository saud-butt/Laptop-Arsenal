# Laptop-Arsenal

To start back-end install the following dependencies first using npm from command line

1)Bcrypt
2)body-parser
3)cors
4)express
5)gravatar
6)jsonwebtoken
7)lodash
8)mongoose
9)mongoose-aggregate-paginate-v2
10)mongoose-autopopulate
11)morgan-body
12)multer
13)multer-s3
14)passport
15)passport-jwt
16)validator

then start back-end with 'npm run server' command.
